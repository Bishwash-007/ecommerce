## Run Locally

Clone the project

```bash
  git clone https://github.com/Bishwash-007/ecommerce.git
```

Go to the project directory

```bash
  cd ecommerce
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
